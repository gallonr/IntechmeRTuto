# IntechmeRTuto 0.1.0

## Première version

* Package initial avec 18 tutoriels interactifs learnr
* Fonctions `list_tutorials()` et `run_tutorial()`
* Documentation complète pour étudiants et enseignants

## Tutoriels inclus

### Niveau Débutant (1-4)
* 01-objets-base : Vecteurs, tibbles et structures de données
* 02-import-donnees : Importation et exploration de données
* 03-dplyr-base : Opérations de base avec dplyr
* 04-dplyr-mutate : Création et transformation de variables

### Niveau Intermédiaire (5-12)
* 05-agregation : Group by et summarize
* 06-tidyr : Restructuration de données avec tidyr
* 07-stringr : Manipulation de chaînes de caractères
* 08-forcats : Gestion des facteurs
* 09-lubridate : Manipulation de dates et heures
* 10-ggplot2-intro : Introduction à ggplot2
* 11-ggplot2-univarie : Visualisations univariées
* 12-ggplot2-bivarie : Visualisations bivariées

### Niveau Avancé (13-16)
* 13-ggplot2-custom : Personnalisation de graphiques
* 14-ggplot2-avance : Techniques avancées ggplot2
* 15-workflows : Workflows complets d'analyse
* 16-purrr : Programmation fonctionnelle

### Bonus (17-18)
* 17-plotly : Visualisations interactives
* 18-cartographie : Cartographie avec R

## Infrastructure
* CSS personnalisé pour l'interface
* Support pour images et logos
* Structure compatible avec les packages R standards
